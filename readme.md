# API Automation Framework

This is an API automation framework built using Java, Maven, RestAssured, and TestNG.

## Prerequisites

- Java 8 or higher
- Maven 3.6.0 or higher
## How to Run 
- Add all maven dependency from pom.xml
- Run testng.xml file
